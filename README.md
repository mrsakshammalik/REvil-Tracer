# Installation
Instructions on how to install *TheFatRat*
```bash
git clone https://github.com/mrsakshammalik/REvil-Tracer.git
cd REvil-Tracer
chmod +x setup.sh && ./setup.sh
```
### Update 
```bash
cd REvil Tracer
./update && chmod +x setup.sh && ./setup.sh
```
### Troubleshoot on REvil-Tracer
chk_tools script to use in case of problems in setup.sh of REvil-Tracer
this script will check if everything is in the right version to run REvil-Tracer
and will also provide you a solution for the problem
```
cd REvil-Tracer
chmod +x chk_tools 
./chk_tools
```
